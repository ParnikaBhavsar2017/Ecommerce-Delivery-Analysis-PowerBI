📊 E-commerce Delivery Performance Analysis (Power BI)
📌 Project Overview
This project analyzes e-commerce order data to evaluate delivery performance across different cities and identify potential locations for new distribution hubs.
The goal is to support business decision-making using data-driven insights.
🎯 Business Problem
The company wants to improve delivery speed and customer satisfaction.
Key questions:
Which cities generate the highest order volume?
Which cities have higher average delivery times?
Do cities with hubs perform better?
Where should new hubs be opened?
📈 Key Insights
Cities with hubs show faster average delivery time (1–2 days).
Non-hub cities experience slower delivery (4–6 days).
High-demand cities without hubs are strong candidates for expansion.
Strategic hub placement can reduce delivery delays and improve customer experience.
